## Deviations & learnings

- **`--surface-hero` stayed a light cream** (plan draft had charcoal): `/apply`, auth pages and other art-heroes still consume it with dark text mid-migration; the v3 Home hero uses its own photo+scrim instead.
- **CSS-comment gotcha:** `ws-*/adm` inside a CSS comment terminated it early (`*/`) and the optimizer silently dropped the entire remap rule — caught only because verification grepped the **compiled** CSS. Always verify the built stylesheet, not the source.
- The pixel-faithful mark PNG beats a hand vector for apple-icon/OG (composite the asset); the favicon still wants a simplified vector for 16px legibility.
- Filenames with trailing spaces (OneDrive drops) must be renamed via git-bash, not PowerShell.
